# git 사용법

> ## git init 해당폴더 깃 초기화 (초기 세팅?!)

> ## git remote add origin "주소" - 해당 주소로 오리진 연결

> ## git add . / git commit -m '커밋 메시지' / git push origin '브랜치명' - 작업한 결과물을 저장 후 연결된 github 주소로 올리기!

const isEven = () => {
  const result = [];
  for (let i = 1; i <= 50; i++) {
    if (i % 2 === 0) {
      result.push(i);
    }
  }
  return result;
};
